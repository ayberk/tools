# Minimalist RSS Reader

A clean, distraction-free RSS reader. Built for simplicity.

![RSS Reader Screenshot](https://share.cleanshot.com/N8fg7Nzw)

## Features

- **Instant Fetch**: Paste any RSS feed URL and read immediately.
- **Pagination**: Browse through feed items comfortably with built-in pagination.
- **Responsive**: Works beautifully on desktop and mobile.
- **Privacy Focused**: No servers, no accounts, no cookies.

## Technologies

- **HTML5**: Semantic and SEO-optimized.
- **CSS3**: Vanilla CSS with modern flexbox/grid layouts and distinct variables.
- **JavaScript**: Vanilla JS using the [RSS2JSON](https://rss2json.com/) API to parse feeds.

## How to Use

### 1. Open and Run
Simply double-click `index.html` to open it in your browser. No server or installation required!

### 2. Enter a Feed
Try these feeds to get started:
- `https://ayberkyilmaz.net/index.xml` (This blog)
- `https://news.ycombinator.com/rss` (Hacker News)
- `https://www.theverge.com/rss/index.xml` (The Verge)
